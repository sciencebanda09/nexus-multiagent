# Code of Conduct

## Our Standards

- Be respectful and constructive
- Welcome beginners — everyone started somewhere
- Focus feedback on code and ideas, not people
- No harassment, discrimination, or personal attacks

## Enforcement

Violations can be reported to coderperson043@gmail.com.
Maintainers reserve the right to remove comments or ban contributors who violate these standards.

## Scope

Applies to all project spaces: GitHub issues, PRs, discussions, and any community channels.
